# aapanel_chinese

#### 介绍
基于aapanel 6.8.21 制做的中文包

#### 降级命令
由于汉化包更新太过麻烦，因此一般都是抽空更新。
如果有新版本无法使用，可用本命令降级到6.8.21版本。           
curl https://raw.githubusercontent.com/gacjie/aapanel_chinese/main/update6_en.sh|bash        

注意：降级脚本已经增加安装中文包功能，因此无需使用安装教程再次安装。

#### 安装教程
1.  下载文件包解压文件
2.  将panel文件夹打包
3.  上传到/www/server目录下
4.  解压文件包覆盖文件
5.  重启宝塔面板
